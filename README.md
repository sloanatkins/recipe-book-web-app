# Recipe Book Web App

A web-based recipe management application built with HTML, CSS, JavaScript, and Firebase.  
This project is actively under development and is being expanded incrementally.

## Overview
The Recipe Book Web App allows users to create, store, and organize recipes through a clean, intuitive interface. The project focuses on building a lightweight, maintainable full-stack web application with real-time data storage and user authentication.

## Tech Stack
- HTML, CSS, JavaScript  
- Firebase Authentication  
- Firebase Firestore  
- Firebase Hosting  

## Current Features
- User authentication with Firebase  
- Create and store recipes in a cloud database  
- Structured recipe data (title, ingredients, instructions)  
- Clean, responsive UI layout  
- Persistent data across sessions  

## Planned Features
- Recipe categories and tags  
- Search and filtering  
- Favorites and ratings  
- Image uploads for recipes  
- Improved mobile responsiveness  
- Public recipe sharing
