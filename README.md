# Recipe Book Web App

A web-based recipe management application built with HTML, CSS, JavaScript, and Firebase.  
This project is currently under active development and is being expanded incrementally.

## Overview
The Recipe Book Web App allows users to create, store, and organize recipes in a clean, easy-to-use interface. The goal of the project is to build a lightweight, maintainable full-stack web application with real-time data storage and authentication.

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting (planned)

## Current Features
- User authentication with Firebase
- Create and store recipes in a cloud database
- Structured recipe data (title, ingredients, instructions)
- Clean, responsive UI layout
- Persistent data across sessions

## Planned Features
- Recipe categories and tags
- Search and filtering
- Favorites and ratings
- Image uploads for recipes
- Improved mobile responsiveness
- Public sharing of recipes

## Status
This project is **still in progress**.  
New features and refinements are being added iteratively as part of ongoing development.
