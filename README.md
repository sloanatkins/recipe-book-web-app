# Recipe Book Web App

A modern, web-based recipe management application built with **HTML, CSS, JavaScript, and Firebase**.  
Designed to help users organize, store, and access family recipes through a clean, intuitive interface.

This project is actively under development and is being expanded incrementally.

---

## Overview

The Recipe Book Web App allows users to securely create, store, and manage recipes in a cloud-backed environment. The application emphasizes simplicity, usability, and maintainability while leveraging Firebase for real-time data storage and authentication.

The long-term goal is to evolve this into a fully featured digital cookbook with social sharing, personalization, and export capabilities.

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Services:** Firebase Authentication, Firebase Firestore  
- **Hosting:** Firebase Hosting  

---

## Current Features

- Secure user authentication with Firebase  
- Create, edit, and store recipes in a cloud database  
- Structured recipe data (title, ingredients, instructions)  
- Clean, responsive UI  
- Persistent data across user sessions
